VS:
- Install C# + asp.net web app (MVC)

- Install Newtonsoft.json NuGet
- Install Color Preview Extension



1. Install VS Code 2022
2. Publish Website to folder
3. Turn windows features on or off > Enable features: https://youtu.be/moqe9mh3j2c
4. Install ASP.NET Core 6.0 Runtime - Windows Hosting: https://dotnet.microsoft.com/en-us/download/dotnet/thank-you/runtime-aspnetcore-6.0.10-windows-hosting-bundle-installer
5. Add folder in IIS (port 80 = HTTP; port 443 = HTTPS: https://youtu.be/kn_KUWOpMdM
6. set Inbound request for port
7. claim subdomain (new google account?): https://freedns.afraid.org/domain/registry/
8. DNS A record to link the URL to the public IP of the server
9. IIS > Bindungen Bearbeiten > Hostname eingeben
10. Website müsste theoretisch öffentlich zugänglich sein.
